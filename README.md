# bevy_mod_rounded_box

This crate provides a rounded box shape for Bevy.

## Dependency

```toml
[dependencies]
bevy_mod_rounded_box = "0.2"
```

## Example

A rotating rounded cube with a draughts board texture.

```shell
cargo run --example cube
```
